# 🎂 Kinza Birthday

An interactive birthday surprise web page built for Kinza — featuring a multi-step unlock animation, animated cake with a blowable candle, confetti, floating hearts, a memory lane carousel, and a toggleable party-mode theme.

## 🌐 Demo

[Live Site on GitHub Pages](https://codwithtayyab96.github.io/Kinza-birthday/)

## 🚀 Quick Start

No build step is required. Open `index.html` directly in your browser, or serve the project root with any static file server:

```bash
# Using Python
python3 -m http.server 8080
# Then open http://localhost:8080 in your browser
```
